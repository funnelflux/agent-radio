# Code of Conduct

Agent Radio should be a practical, respectful project for users and contributors.

## Expected behavior

- Be kind, direct, and constructive.
- Assume good intent, but accept corrections.
- Keep discussion focused on the project.
- Respect different experience levels and workflows.
- Report security concerns privately, not in public issues.

## Unacceptable behavior

- Harassment, threats, insults, or personal attacks
- Discriminatory language or behavior
- Publishing private information without permission
- Spam, trolling, or repeated off-topic comments
- Pressuring the maintainer or contributors for unpaid work

## Enforcement

The maintainer may edit, hide, or delete comments; close issues or pull requests; block users; or take any other action needed to keep the project healthy.

This is a solo-maintainer project. The maintainer makes final calls on enforcement.

## Reporting

Report conduct concerns privately through the maintainer's GitHub profile.

If the concern is also a security issue, follow `SECURITY.md`.
